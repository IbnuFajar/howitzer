Howitzer

Ruby based framework for acceptance testing

View on GitHub View on RubyGems Api Documentation

Have you been tired to spend a lot of time, money and efforts for acceptance test framework development? Do you want to concentrate on automated testing immediately? There is a good solution - Howitzer.

Howitzer allows to generate and configure test framework for 5 minutes. It uses the best practices and design patterns. It has built-in configurations for parallel cross-browser testing in the Cloud.

Do not invent a wheel! Just start using Howitzer and make it easy on yourself!

Key Benefits

Install Howitzer

Ruby gem installation:

$ gem install howitzer

New project folder creation:

$ mkdir test_automation

Navigation to project folder:

$ cd test_automation

Project skeleton generation:

$ howitzer install --rspec --cucumber
